# ⚙️ STM32 Servo Brake Control System

An embedded systems project built using STM32 microcontrollers to control servo-based braking mechanisms through PWM control.

---

## 🚀 Features

- 🔧 PWM-based servo control
- ⏱️ Real-time motor actuation
- 📟 STM32 HAL integration
- 🧠 Low-level C implementation
- ⚡ Hardware-software interfacing

---

## 🛠️ Tech Stack

- C
- STM32CubeIDE
- STM32 HAL
- Embedded Systems Programming

---

## 🧠 Project Overview

This project demonstrates low-level microcontroller programming and hardware integration.  
The system controls servo-based brake actuation using PWM signals generated by STM32 timers.

---

## 🧪 Key Learning Outcomes

- Timer configuration
- Interrupt handling
- Embedded debugging
- Hardware interfacing
- Real-time control logic

---

## 🧩 Hardware Used

- STM32 Microcontroller
- Servo Motor
- External power supply
- Breadboard + wiring setup

---

## 🚀 Build Instructions

1. Open project in STM32CubeIDE
2. Configure target board
3. Build and flash firmware
4. Connect servo to PWM pin
